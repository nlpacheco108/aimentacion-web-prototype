# AImentacion Web Prototype

Prototipo estatico de la plataforma AImentacion para bancos de alimentos y entidades sociales.

## Contenido

- `index.html`: landing y demo principal
- `styles.css`: estilos de interfaz
- `script.js`: comportamiento del selector de entidades piloto
- `AImentacion_portada_v2.svg`: ilustracion de portada

## Uso

Abrir `index.html` en un navegador moderno.
